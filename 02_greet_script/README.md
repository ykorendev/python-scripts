

# Greet script

## Description

This script greets the user by name using a command-line argument.

## Usage
```bash
python3 greet.py <name>
```
## Example
```bash
pyton3 greet.py Yarin
```

**Output:**
```bash
Hello, Yarin! Welcome to Python scripting!
```

## Notes
If no name is provided, the script displays usage instructions
