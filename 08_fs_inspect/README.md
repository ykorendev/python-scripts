

# Filesystem Inspect Script

## Description
Inspect a directory:
- count files
- calculate total size
- show disk usage

## Usage
```bash
python3 fs_inspect.py 
python3 fs_inspect.py <directory path>
```
**Output:**
Directory: <directory path>
Files: <number of files>
Total size: <size of directory> MB
Disk free: <free disk space> GB

## Notes
Default directory is current directory.
